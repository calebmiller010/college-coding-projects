To compile a program: 
    1. Change into the directory that the file you wish to compile is in (use 'cd' command to change directory, followed by a space, and the name of the directory)
        example: type 'cd prob1' and press return
                 type 'cd prob1a' and press return

    2. Compile the program using the command 'javac [filename.java]'
        example: type 'javac LinkedCircleList.java' and press return


To run a program:
    After the program has been compiled, use the command 'java [filename]' to run the program

        example: type 'java LinkedCircleList' and press return. Then the program will run
